Closing brackets should match the indentation of the opening bracket.

### Anti-pattern

```python
result = function_that_takes_arguments('a', 'b', 'c',
                                       'd', 'e', 'f',
)
```

### Best practice

```python
result = function_that_takes_arguments('a', 'b', 'c',
                                       'd', 'e', 'f',
                                       )
```

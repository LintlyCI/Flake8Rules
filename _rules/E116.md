Comments should be indented relative to the code in the block they are in.

### Anti-pattern

```python
    # 'httpd/unix-directory'
mimetype = 'application/x-directory'
```

### Best practice

```python
# 'httpd/unix-directory'
mimetype = 'application/x-directory'
```

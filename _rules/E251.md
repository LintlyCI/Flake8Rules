There should be no spaces before or after the `=` in a function definition.

### Anti-pattern

```python
def func(key1 = 'val1',
         key2 = 'val2'):
    return key1, key2
```

### Best practice

```python
def func(key1='val1',
         key2='val2'):
    return key1, key2
```

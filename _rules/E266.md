links:
  - https://www.python.org/dev/peps/pep-0008/#block-comments
---

There should be only one leading `#` for a block comment.

### Anti-pattern

```python
## Prints hello
print('hello')
```

### Best practice

Only use one `#`.

```python
# Prints hello
print('hello')
```
